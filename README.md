# Astral-Link

Astral-Link is a Full Stack Chatting App. Uses Socket.io for real-time communication and stores user details in encrypted format in Mongo DB Database.

## Tech Stack

**Client:** `React JS`

**Server:** `Node JS`, `Express JS` & `Socket.io`

**Database:** `Mongo DB`

- ### Real-Time Chatting
- ### One-to-One Chat
